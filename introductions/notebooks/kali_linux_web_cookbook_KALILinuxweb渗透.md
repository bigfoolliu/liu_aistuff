## 2.侦查

使用nmap来扫描

```sh
# 1.查看服务器是否响应
nmap -sn 192.168.56.182

# 2.查看哪些端口打开了
nmap 192.168.56.182

# 3,查看运行的服务的版本并才猜测运行的操作系统版本
nmap -sV -O 192.168.56.182
```
